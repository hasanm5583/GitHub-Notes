# GitHub-Notes
Repo for my Github notes
## Git
You can control the version history of you Github repository being able to change one specific edit in the file without having to change all the edits 

## Github
Github is the server that Git runs on!

## Repo

A repo or repository is a Big project or Big file on Github. That can have many files on one repository So all your files can be organized based on the repositores topic 

## Commit 
The save button that saves all the edits or changes that you have made on a file in a repo. You can also add comments. To remember what you changed. You can also go back to your commits history to  see what changes you made in each commit. There you can see what change you made in a certain commit and undo only that certain commit without having to undo every edit history like in Google docs.
